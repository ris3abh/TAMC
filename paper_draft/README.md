# TAMC Paper Draft

This folder contains the first workshop-paper draft for TAMC
(Topological Attractor Meta-Control).

## Contents

- `tamc_workshop_draft.md` — the draft manuscript.

## Status

This is a **first draft, not submission-ready**. It is a conservative,
structured assembly of existing repo evidence, written to expose gaps
clearly rather than to overclaim.

## Source material

The draft is assembled from, and should stay consistent with:

- `paper_notes/introduction.md`
- `paper_notes/problem_statement.md`
- `paper_notes/methodology.md`
- `paper_notes/related_work.md`
- `paper_notes/research_brief.md` (especially Section 19, "Current
  Empirical Status," which is the running log of actual results)
- the root `README.md`

Every numeric claim in the draft should be traceable to a file under
`figures/` or to `paper_notes/research_brief.md`. If a future repo result
changes, update `paper_notes/` first, then bring the draft in line with
it — not the other way around.

## Figures

Figures referenced in the draft are generated by the experiment scripts
in `experiments/` and saved to `figures/` (not tracked in git; regenerate
by re-running the relevant script). The draft currently embeds two real
figure references directly:

- `figures/topology_ablation_heatmap.png`
- `figures/runtime_benchmark.png`

Two figures are still placeholders pending dedicated diagram work (the
pipeline schematic and the detection-curve/adaptation-tradeoff plots) —
search the draft for `TODO`.
