# TAMC Paper Draft

This folder contains the first workshop-paper draft for TAMC
(Topological Attractor Meta-Control).

## Contents

- `tamc_workshop_draft.md` — the draft manuscript.

## Status

This is a **first draft, not submission-ready**. It is a conservative,
structured assembly of existing repo evidence, written to expose gaps
clearly rather than to overclaim.

## Source material

The draft is assembled from, and should stay consistent with:

- `paper_notes/introduction.md`
- `paper_notes/problem_statement.md`
- `paper_notes/methodology.md`
- `paper_notes/related_work.md`
- `paper_notes/research_brief.md` (especially Section 19, "Current
  Empirical Status," which is the running log of actual results)
- the root `README.md`

Every numeric claim in the draft should be traceable to a file under
`figures/` or to `paper_notes/research_brief.md`. If a future repo result
changes, update `paper_notes/` first, then bring the draft in line with
it — not the other way around.

## Figures

All seven figures referenced in the draft are generated by
`experiments/paper_figures.py` and saved to `figures/` (not tracked in
git; regenerate with `uv run python experiments/paper_figures.py`):

- `figures/paper_pipeline_schematic.png` — Figure 1, the closed-loop
  schematic (no data dependency).
- `figures/paper_detection_summary.png` — Figure 2, detection AUROC by
  method across the three controlled systems.
- `figures/paper_adaptation_tradeoff.png` — Figure 3, Net Adaptation
  Score for the original synthetic/ETTh1 results plus the dynamical
  extension.
- `figures/paper_ablation_heatmap.png` — Figure 4, the paper version of
  the topology ablation heatmap.
- `figures/paper_runtime_summary.png` — Figure 5, the paper version of
  the runtime benchmark plot.
- `figures/paper_dynamical_adaptation_summary.png` — Figure 6 (optional,
  referenced in Section 5.2), per-system/per-variant Net Adaptation Score
  for the dynamical forecast-adaptation extension.
- `figures/paper_regime_benchmark_summary.png` — Figure 7 (optional,
  referenced in Section 5.3), TAMC vs. the RG-style regime-similarity gate
  (plus autocorrelation/spectral) across the five real-data benchmark
  datasets.

`experiments/paper_figures.py` reads existing multi-seed summary CSVs
under `figures/` and does not rerun any experiment by default except
lightweight ones (a few seconds to ~70s) if missing; it never
auto-reruns the ~20-minute topology ablation, printing the command to run
manually instead if that summary is absent. No figures are still
placeholders.
